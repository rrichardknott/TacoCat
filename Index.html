Taco Cat Project

'Meow'

The Taco Cat project is a program that demonstrates whether or not a word and or phrase is a palindrome. Palindrome words are those which read the same way when written backwards. For example, the words 'Taco Cat' when spelled backwards become Taco Cat. The same for 'Nurses Run'. When this happens, the word and/or phrase is by definition a palindrome.
